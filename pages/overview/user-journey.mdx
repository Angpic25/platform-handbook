# User Journey

This is the typical flow a client follows when using the platform to manage accessibility issues.

<Steps>

### Land on the Home dashboard

After logging in, the client sees the company level overview with key metrics (active critical issues, resolved issues) and a list of highest impact touchpoints.

![Home dashboard showing company overview with key metrics and highest impact touchpoints](/images/video-home-dashboard-tour.gif)

### Navigate to a Touchpoint overview

The client clicks on a touchpoint from the Home dashboard or from the Touchpoints list to see the detailed compliance dashboard.

![Navigation from home to touchpoint overview](/images/video-home-to-overview.gif)

### Review the Issues list

From the Touchpoint overview, the client switches to the Issues tab to see all identified accessibility issues with filters, search, and sorting.

![Issue management view showing the issues table with filters and toolbar](/images/video-issue-management.gif)

### Open an Issue detail

The client clicks on an issue to open the side panel, which shows the Overview tab with full details, the Fix tab with suggested remediation, and the Jira tab for ticket linking.

![Issue detail side panel with Overview, Fix, and Jira tabs](/images/video-issue-detail-tabs.gif)

### Create a Jira ticket

From the Jira tab in the issue detail, the client can link the issue to an existing Jira ticket or create a new one to assign the fix to a developer.

![Creating a Jira ticket from the issue detail panel](/images/video-jira-from-issue.gif)

### Fix the issue and request a retest

After the development team implements the fix, the client marks the issue as Fixed and requests an expert retest to validate the remediation.

![Flow showing how to fix an issue and request an expert retest](/images/video-fix-and-retest.gif)

### Track progress over time

The client returns to the Touchpoint overview to monitor the trend chart and compliance metrics, verifying that the overall accessibility score improves over time.

![Touchpoint overview trend chart showing progress over time](/images/placeholder-trend-chart.png)

</Steps>
