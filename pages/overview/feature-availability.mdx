# Feature Availability by Tier

The features available to each client depend on the tier purchased for their touchpoint. Use this page as a reference when preparing for a delivery call, answering client questions, or configuring a new account.

When a feature is not available for a given tier, it is simply hidden from the client's interface — they will not see an error or a locked state, just no access to that section or functionality.
