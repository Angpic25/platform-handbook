# Touchpoints List

The Touchpoints list page shows all touchpoints (websites and digital applications) configured in the client's account.

![Touchpoints list showing all configured touchpoints with domain, audit date, compliance, and total issues](/images/placeholder-touchpoints-list.png)

## What the client sees

- **Domain** for each touchpoint.
- **Audit date** showing when the last accessibility audit was performed.
- **Compliance percentage** against WCAG standards.
- **Total issues** identified for each touchpoint.

The client can click any row to open the Touchpoint overview. Pagination is available with a default of 25 rows per page.

<Callout>
The touchpoints list is populated by AccessiWay after each audit. The client cannot add or remove touchpoints from this view.
</Callout>
