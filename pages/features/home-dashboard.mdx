# Home Dashboard

The Home dashboard is the first screen the client sees after logging in. It provides a company level overview of accessibility status across all touchpoints.

<video aria-hidden="true" src="/images/video-home-dashboard-tour.mp4" autoPlay loop muted playsInline style={{ width: '100%', borderRadius: '8px' }} />

## What the client sees

- **Welcome banner** with a summary of the three main platform capabilities: identify and prioritize issues, coordinate fixes, and view reports in one place.
- **Active critical issues** count showing how many critical issues remain open across all touchpoints, with the total issue count for context.
- **Resolved issues** count showing how many issues have been expert validated, reinforcing progress.
- **Highest impact touchpoints** table listing the top 3 touchpoints by severity, with compliance percentage and open issue count for each.
- **"View all touchpoints"** button to navigate to the full touchpoints list.

<Callout>
The Home dashboard is read only. The client cannot edit data here, only navigate to touchpoints for deeper analysis.
</Callout>
