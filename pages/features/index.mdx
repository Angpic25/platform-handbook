## Feature summary

| Feature | What it does |
|---------|-------------|
| **Home Dashboard** | Company level overview with key metrics and highest impact touchpoints |
| **Touchpoints List** | Browse and navigate all touchpoints in the account |
| **Touchpoint Overview** | Compliance metrics, trend chart, user impact, and user flows for a single touchpoint |
| **Issue Management** | Filter, sort, search, and export accessibility issues |
| **Issue Detail** | Full issue information with Overview, Fix, and Jira tabs |
| **Integrations** | Connect with Jira, Trello (coming soon), Azure DevOps (coming soon) |
| **Settings and Team** | Security (2FA) and team member management |
| **Platform Guide** | In platform onboarding, feature explanations, and FAQ |
