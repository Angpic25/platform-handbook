## Key FAQ highlights

| Question | Short answer |
|----------|-------------|
| What is this platform showing me? | A dynamic dashboard with compliance score, issues, severity, and remediation progress |
| What's the difference between "fixed" and "resolved"? | Fixed = dev team marked it done. Resolved = expert-validated via retest |
| How does the retest work? | Up to 1 retest per touchpoint. The expert reviews the full touchpoint |
| Does the mitigation widget replace code fixes? | No — it's a bridge solution while you implement permanent fixes |
| How does the Jira integration work? | Connect Jira to sync issues, manage them in your existing workflows |
