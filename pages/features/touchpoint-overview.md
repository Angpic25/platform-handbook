# Touchpoint Overview

Compliance metrics, issue trends, and user-impact breakdown — everything about one touchpoint in a single screen.

## How it works

- **Compliance level** shows overall compliance percentage
- **Issue counters** break down into Open (purple), Fixed (pink), Resolved (green), and Dismissed (grey)
- **Issue trend chart** shows how issues evolve over time — toggle between area chart, pattern chart, and data table
- **Issues by user impact** splits issues into Critical, High, and Low with percentages
- **User flows to prioritize** lists flows with the highest concentration of critical issues
