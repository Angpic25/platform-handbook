# Integrations

Push accessibility issues to Jira with severity and WCAG references — keep everything in sync.

![Integrations page](/images/placeholder-integrations-page.png)
<!-- 📸 CAPTURE NEEDED: Integrations tab showing Jira Cloud (available) and Trello (coming soon) cards -->

## How it works

- **Jira Cloud** — available now. Creates tickets with severity and WCAG references, syncs status bidirectionally
- **Trello** — coming soon. Will turn issues into Trello cards for board-based tracking
- Each touchpoint has its own Integrations tab with a **Configure** button for Jira

## Step by step

### 1. Go to the Integrations tab

Open any touchpoint and click the **Integrations** tab.

![Integrations tab](/images/placeholder-integrations-tab.png)
<!-- 📸 CAPTURE NEEDED: Touchpoint detail with Integrations tab selected -->

### 2. Configure Jira Cloud

Click **Configure** on the Jira Cloud card and follow the setup flow.

<!-- 🎥 VIDEO NEEDED: Jira Cloud configuration flow from clicking Configure to completion, ~25 seconds -->
*[Video: Setting up Jira Cloud integration]*

### 3. Verify the connection

Open any issue's **Jira** tab to confirm the linked ticket appears.

![Jira connected](/images/placeholder-jira-connected.png)
